# DATMM Schema (see [Change Log](CHANGELOG.MD))

This repository tracks released versions of the DATMM schema and their associated source spreadsheets and RDF exports.

## Layout

- `schema/<version>/excel/` — the Excel spreadsheet used as the source of truth for that version
- `schema/<version>/rdf/rdfxml/` — RDF/XML representation for that version
